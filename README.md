# AnimatingHeaderView

A UIView subclass with animating image content.
Currently only for iOS 9.

## Installation
### Cocoapods
To integrate AnimatingHeaderView into your Xcode project using CocoaPods, specify it in your Podfile:
```ruby
pod 'AnimatingHeaderView'
```
